Introduction

In this study, Ive chosen to examine the data on fraudulent online payment transactions. Visit https://www.kaggle.com/datasets/jainilcoder/online-payment-fraud-detection to access the information. Building a model that can identify fraudulent transactions is the aim of this project. Utilizing the ROCcurve, also known as the Receiver Operating Characteristic, I will be using the machine learning model Logistic Regression to compare their accuracy. We start by importing the required libraries and loading the datasets data.
